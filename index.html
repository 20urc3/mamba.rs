Analysis Overview

DAG

Eliminates redundant computations in a basic block by sharing common sub‑expressions and tracking data dependencies.

Call Graph

Models inter‑function calls with synthetic nodes for external world to understand reachable and escaping edges.

CFG

Captures intra‑function control flow and identifies loop back‑edges for further analyses.

Dominator Trees

Uses Lengauer–Tarjan to compute domination and post‑domination, enabling must‑execute reasoning.

Data‑Flow Solver

Iterative work‑list engine supporting liveness, reaching definitions, available expressions and more.

Alias / Points‑to

Inclusion‑based Andersen analysis with lazy cycle detection for scalable points‑to sets.

Technology

Mamba is written in Rust and hooks directly into LLVM to parse bitcode files. It transforms low‑level IR into relational facts stored in SQLite, then leverages the Soufflé Datalog engine for ultra‑fast rule evaluation. This hybrid approach couples Rust’s performance with Datalog’s declarative power, giving security researchers instant feedback on program‑wide vulnerabilities—even on million‑line codebases.

Rust + LLVM
